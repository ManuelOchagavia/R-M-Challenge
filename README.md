# Rick and Morty Challenge

## Instrucciones de uso
Para utilizar el repositorio se debe instalar las librerias de node con:

~~~
yarn install
~~~

Para correr el programa utilizar:

~~~
yarn start
~~~

Para correr los tests utilizar: 

~~~
yarn jest
~~~


## Estructura de carpetas

- R-M-CHallenge

    - \_\_tests__
    - src
